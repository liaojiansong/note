## mark down 语法
####    [教程地址](https://blog.huihut.com/2017/01/25/MarkdownTutorial/)
#### 标题篇
* \# 一级标题
* \## 二级标题
* ....(以此类推)

#### 代码篇
* \`这里放代码\`   单行代码
* \```这里放代码``` 代码块(还可以在\`\`\`后面指定代码语言) 如\`\`\` sql \`\`\`

#### 列表篇
* `1.  2. ...` =>有序列表
* `*` => 无须列表

#### 区块片篇
* `>` 一级区块
* `>>` 二级区块

#### 分割线篇
* `***` 分隔方式一
* `---`分隔方式二

#### 链接篇
* `[这里放链接名称](这里放url)` 超链接
* `![这里放图片名](这里放ur)` 图片

### 表格篇
* 绘制表格
```markdown
    |这里放表头|表头|
    |---:|---:|
    |这里放数据|数据|
```

#### 技巧篇
* `连续两个空格然后回车` 换行
